# Incidents

Postmortems tecnicos de bugs e incidentes confirmados.

Ainda nao ha incidentes registrados. Crie quando um bug de producao for investigado e resolvido.

Use `templates/incident.md` como base.

## Regras

- Nao incluir credenciais, tokens ou dados pessoais.
- Registrar causa raiz somente quando confirmada.
- Linkar para mapas de feature e workflows relevantes.

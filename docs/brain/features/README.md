# Features

Mapas progressivos por feature. Cada arquivo documenta pontos de entrada, componentes, hooks, fluxo e riscos de uma area funcional.

## Features mapeadas

- `kanban-dashboard.md`: Kanban principal, esteiras, drag-and-drop, filtros, bulk update.
- `process-detail.md`: detalhe de processo, documentos, movimentos, atividades, insights.
- `auth-users.md`: login, autenticacao, roles e sessao.

## Quando criar um novo mapa

Crie um novo arquivo quando uma task revelar area funcional recorrente sem mapa proprio. Use `_template.md` como base.

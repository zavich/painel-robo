# Workflows

Roteiros para tarefas recorrentes de investigacao, debug e mudanca.

## Workflows disponiveis

- `investigacao-progressiva.md`: protocolo geral para investigar areas desconhecidas.

## Quando criar um novo workflow

Crie quando uma tarefa recorrente exigir mais de 3 passos e envolver multiplos arquivos. Use `_template.md` como base.
